# Extract PDF Pages by Keywords

Find and extract PDF pages containing one or two keywords or text patterns. Save matching pages to a new PDF privately in your browser, with original layout preserved.

![Watch demo](Screenshot.png)](demo.mov)


## Live tool

https://vgk575.github.io/pdf-text-extractor/

## Features

- Find pages by keyword or text pattern
- Support for one or two search patterns
- Extract matching pages into a separate PDF
- Preserve the original page layout
- Run entirely in the browser for privacy
- No upload, no registration
